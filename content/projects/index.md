1. **STYLE-TRANS-FAIR** : codabench challenge
    - A competition about bias in images. My classmates and I created a dataset where bias can be visualised. Your task is to create robust systems that can classify the images without learning the features of the bias induced by the style transfer. The train set will be split such that each class will have a dominant style. The test set, on the other hand, will be balanced in order to see how well you managed to overcome the bias. For more information, check the Github repository.
    - Here [the Link to the competition](https://www.codabench.org/competitions/601/) and [link to github repository](https://github.com/fnachalearn/style-trans-fair)

3. **Punctuation restoration** project (see repo for more details, there will be presentation powerpoint there)
    - The aim was to train models capable of restoring punctuation to unpunctuated text.
    - the use of 2 articles([Automatic punctuation restoration with BERTmodels](https://arxiv.org/pdf/2101.07343.pdf) and [Punctuation Restoration using Transformer Models for High-and Low-Resource Languages](https://aclanthology.org/2020.wnut-1.18.pdf))to implement the SOTA in this area. 
    - use of Cbow and Bert as embedding layer and as classifier a bidirectional lstm, then a transformer and directional lstm.
